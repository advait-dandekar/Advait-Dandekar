My Projects

Voice Controlled Robot Car using Raspberry Pi Pico W

Used a Bluetooth Module for data communication between smartphone and Raspberry Pi. Voice Commands were given through a free-to-use app on the smartphone. The movements in the car would then be reflected through a motor driver board to control the motors. Raspberry Pi was the brain of this project.

Components required:
  1. Bluetooth Module
  2. Raspberry Pi
  3. Motor Driver Board
  4. Gear Motors
  5. Ultrasonic Sensors

Motion Sensor Based Lights using Arduino

Used two PIR Motion sensors, an Arduino Uno, a relay module, and a light bulb to facilitate easy switching on/off of light bulb whenever a person entered the staircase.

Components required:
  1. PIR Motion Sensors
  2. Arduino Uno
  3. Single Channel Relay Module
  4. Light bulb
